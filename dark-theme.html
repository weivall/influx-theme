Icon Font

Icon font created using icomoon.io

Font files are referenced from bootstrap-theme.min.css as a relative path ../fonts/. To ensure proper display, make sure your folder structure is like this:

css/
    bootstrap.min.css
    bootstrap-theme.min.css
    bootstrap-theme.min.css.map
fonts/
    icomoon.eot
    icomoon.svg
    icomoon.ttf
    icomoon.woff
    icomoob.woff2

How to use:

The icon class can be applied to any element, but realistically works best when applied to inline elements such as <span>

Collapsible Panels

How to Use

Collapsible panels are a combination of some custom CSS and Bootstrap's built in Collapse javascript. This component is useful when you want to list multiple items in rows, and each item has a too much attached information to show all at once.

Collapsible panels can behave one of two ways:

  1. Default Style - Collapsible panels can be opened / closed separately of each other.
  2. Accordion Style - When only one collapsible panel in a list can be open at one time. Opening a panel closes all other open panels.

Implementation

Since the collapse functionality is built-in, this is pretty straightforward. Collapse works by wrapping an element with a collapsing <div> and assigning it a trigger via custom HTML attributes. In these collapsible panels, div.panel-body gets wrapped.

<div class="panel panel-collapse">
  <div class="panel-heading">
    <h2 class="panel-title">
      <a role="button" data-toggle="collapse" class="collapsed" href="#panel-id-here">
        <span class="caret"></span> Click to Toggle
      </a>
    </h2>
  </div>
  <div class="collapse" id="panel-id-here" style="height: 0px;">
    <div class="panel-body">
      ...
    </div>
  </div>
</div>

Adding style="height: 0px;" sets the collapsible panel to be closed by default

Accordion Style

This effect can be achieved by simply wrapping all your collapsible panels with div.panel-group and adding the data-parent attribute to collapse element.

<div class="panel-group" id="panel-group-id" role="tablist">
  <div class="panel panel-collapse">
    <div class="panel-heading">
      <h2 class="panel-title">
        <a role="button" data-toggle="collapse" class="collapsed" data-parent="#panel-group-id" href="#panel-id-1">
          <span class="caret"></span> Click to Toggle
        </a>
      </h2>
    </div>
    <div class="collapse" id="panel-id-1">
      <div class="panel-body">
        ...
      </div>
    </div>
  </div>

  <div class="panel panel-collapse">
    <div class="panel-heading">
      <h2 class="panel-title">
        <a role="button" data-toggle="collapse" class="collapsed" data-parent="#panel-group-id" href="#panel-id-2">
          <span class="caret"></span> Click to Toggle
        </a>
      </h2>
    </div>
    <div class="collapse" id="panel-id-2">
      <div class="panel-body">
        ...
      </div>
    </div>
  </div>

  <div class="panel panel-collapse">

</div>

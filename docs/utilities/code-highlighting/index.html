Code Highlighting

How to Use

There are many code highlighting JS libraries out there, for example we use Prism.js to handle code highlighting in our Developer Docs. There is no built-in code highlighting within Bootstrap, so I have created a set of highlighters for manual use. In the future there will be a code highlighting feature built into the theme.

Concept
In HTML there are a variety of tags specifically for formatting text. Most websites and applications no longer use these tags as the same effect can be achieved with CSS more cleanly. I have repurposed these tags to function as code highlighters, scoped within code samples.

Taking into consideration the possibility that a external code highlighter might be used in conjunction with this theme, I have put a failsafe in play:

For Code Highlighting to take effect, just add a class to your sample,

<pre class="rainbow-highlighter">
  <code></code>
</pre>

<code class="rainbow-highlighter"> </code>

Then use these tags to apply highlights:

Bold <b> Obtuse Mystical Being
Important <strong> Obtuse Mystical Being
Italic <i> Obtuse Mystical Being
Emphasized <em> Obtuse Mystical Being
Marked <mark> Obtuse Mystical Being
Small <small> Obtuse Mystical Being
Deleted <del> Obtuse Mystical Being
Inserted <ins> Obtuse Mystical Being
Subscript <sub> Obtuse Mystical Being
Superscript <sup> Obtuse Mystical Being
Underline <u> Obtuse Mystical Being

Example SCSS with Code Highlighting

/*
  Basic Button
  -------------------------------------------------
*/

@transition-time: 0.25s;

.btn {
  height: 32px;
  line-height: 32px;
  border-radius: 4px;
  padding: 0 13px;
  font-size: 15px;
  font-weight: 500;
  background-color: @btn-default-bg;
  border: none;
  text-align: center;
  color: @btn-default-text;

  transition: 
    color @transition-time ease,
    background-color @transition-time ease;

/* Hover State */
&:hover {
  cursor: pointer;
  background-color: @btn-default-bg-hover;
  color: @btn-default-text-hover;
}

/* Color Alternatives */
&.blue {
  background-color: @btn-blue-bg;
  color: @btn-blue-text;

  &:hover {
    background-color: @btn-blue-bg-hover;
    color: @btn-blue-text-hover;
  }
}
}

Buttons

How to Use

Buttons are to software as links are to web sites. They serve to confirm user input, activate tools, change views, etc. Depending on the context you may want a button of a different size or color. In bootstrap there are two ways to make a button:

1. Anchor Tag for when you want a text link to look and behave like a button

<a class="btn btn-default" role="button" href="#">Button Text</a>

2. Button Tag for when you want an actual button

<button class="btn btn-default">Button Text</button>

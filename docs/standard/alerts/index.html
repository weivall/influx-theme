Alerts

Regular Alerts

Cannot be dismissed, can contain links and/or buttons to resolve issue. Usually sits at the top of the page, just below the navbar.

Useful for:

  • User needs to verify email
  • Form validation
  • Confirming a user's action

How to Implement:

<div class="alert alert-success" role="alert">
  Alert text goes here
</div>

Contextual Alternatives:

Success div.alert.alert-success
Primary div.alert.alert-primary
Warning div.alert.alert-warning
Danger div.alert.alert-danger
Info div.alert.alert-info

Dismissable Alerts

Useful when the information or task at hand is low priority. Serves to let the user know then get out of the way.

Useful for:

  • User views a feature for the first time, tutorial text
  • System Updates
  • Confirming a user's action

How to Implement:

<div class="alert alert-success" role="alert">
    Alert text goes here
  <button type="button" class="close" data-dismiss="alert" aria-label="Close">
    <span aria-hidden="true">
      <span class="icon remove"></span>
    </span>
  </button>
</div>

Alerts + Icons

Useful when you want to draw the more attention to the alert, and add a little more context. Any icon from the Icon Font can be used.

How to Implement:

<div class="alert alert-icon alert-success" role="alert">
  <span class="icon alert-triangle"></span>
  Alert text goes here
</div>

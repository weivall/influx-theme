Tabs (Tab Lists)

Useful For:
Tertiary navigation within a page. Best used at the top of the page, to indicate that anything below will be affected by tab-switching

Implementation:

<ul class="nav nav-tablist" role="tablist">
  <li class="active"><a href="#">Active Tab</a></li>
  <li><a href="#">Tab</a></li>
  <li><a href="#">Tab</a></li>
</ul>

Size Alternatives:

Large ul.nav.nav-tablist.nav-tablist-lg
Default ul.nav.nav-tablist
Small ul.nav.nav-tablist.nav-tablist-sm
Extra Small ul.nav.nav-tablist.nav-tablist-xs

Micro Tabs

Useful For:
When you have a smaller component that requires tab functionality. Currently used in the Data Explorer.

Implementation:

<ul class="nav nav-microtabs" role="tablist">
  <li class="active"><a href="#">Active Tab</a></li>
  <li><a href="#">Tab</a></li>
  <li><a href="#">Tab</a></li>
</ul>

Dismissable Tabs
Designed to mimic browser tabs.

NOTE:
Since the dismiss UI is custom, there is no corresponding JS included in the standard Bootstrap package. You will have to write something to handle tab dismissing.
<li>
  <a href="#">
    Tab Label <button class="microtabs-dismiss"></button>
  </a>
</li>

Contextual Alternatives:
Depending on the context in which the tabs are used, you may want to stylize them differently. There are options to match different panel types as well as an option for dark UI.

Dark ul.nav.nav-microtabs.nav-microtabs-dark
Darker ul.nav.nav-microtabs.nav-microtabs-darker
White ul.nav.nav-microtabs.nav-microtabs-white
Spring ul.nav.nav-microtabs.nav-microtabs-spring
Summer ul.nav.nav-microtabs.nav-microtabs-summer
Fall ul.nav.nav-microtabs.nav-microtabs-fall
Winter ul.nav.nav-microtabs.nav-microtabs-winter

Bootstrap Togglable Tabs

This feature is built into `bootstrap.js` and allows for simple toggling between tabs. Bootstrap requires that each instance of a tab bar be initialized separately. Read more at Bootstrap JS Docs - Tabs

The Type Scale

What Is It?

A Type Scale is a ratio that controls the proportions of a corresponding Type System. The Scale is a single number used to calculate leading, font size, and margins for each type element. Using a scale ensures that the type grows in pleasing and consistent proportions.

The Type Scale in this theme is 1.19, which is on the smaller end. Large format print ads use type scales as big as 1.9 for maximum contrast between heading sizes. Some like to use the Golden Ratio as their type scale, but I find it too extreme for Software UI usage.

Things to Consider:
There will always be 6 heading sizes in the browser. It is easiest to pick the Base size (H6) first, then the largest possible size (H1). Adjust the scale until the largest size is equal to your largest.

Why is H6 the same as the Base size?
Since Paragraphs also use the Base size, it may seem natural to make the smallest heading larger. However, most heading styles are bold and the smallest doesn't need a larger font size to be distinct from Paragraphs.

How It Works

See the Scale column for example implementation. Font weights and letter spacing have been hand adjusted for each heading. Values for Final, Line Height, and Margin have all been rounded up to the nearest whole number.

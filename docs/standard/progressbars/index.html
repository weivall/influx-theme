Progress Bars

Basic Bars

Fairly self explanatory, you've seen these before. The only design consideration is showing the progress as a number or hiding it. Depending on the use of the progress bar, having a label can be helpful or cluttering.

Implementation

<div class="progress">
  <div class="progress-bar" style="width: 60%;">
    <span class="sr-only">60%</span>
  </div>
</div>

.sr-only stands for "Screenreader Only" and hides the element such that only screenreaders can see it. This is for accessibility. If you remove the class, the label text will be visible inside the progress bar.

Contextual Alternatives

Success div.progress-bar.progress-bar-success
Primary div.progress-bar.progress-bar-primary
Warning div.progress-bar.progress-bar-warning
Danger div.progress-bar.progress-bar-danger
Info div.progress-bar.progress-bar-info
